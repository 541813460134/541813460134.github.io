git命令

提示

说明：本章内容是博主的Java学习笔记，

git init

git add -A

git commit -m 'di n ci ti jiao'

git remote add origin 远程库

git push -f 远程库

git log //查看日志 git reflog //查看简略日志

git reset --hard 版本号

git branch 分子名 //创建分支 git branch -v 查看分支

git checkout 分支名 //切换分支 git merge 分支名 //把指定的分支合并到当前分支上

git合并冲突

两个分支都做了修改，需要手动合并分支

显示两块代码都做了修改，需要保留自己想要的，然后删掉不需要的和额外加的注释

修改需要提交

合并的分支只会修改合并的分支，不会修改合并过来的分支

创建远程库别名

报错git Unable to access ‘https://github.com/xxxx/xxxx.git/’:OpenSSL SSL_read:Connection was reset 解决方法

git config --global http.sslVerify false
